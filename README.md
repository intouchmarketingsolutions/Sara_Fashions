# Sara Fashion — Luxury E-Commerce

A premium black & gold fashion e-commerce website built with React (Vite).

## Tech Stack
- React 18 + Vite
- Tailwind CSS
- React Router DOM
- Framer Motion
- React Icons

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Pages
- `/` — Home (Hero, Categories, Featured Products)
- `/men` — Men's Collection with filters
- `/women` — Women's Collection with filters
- `/kids` — Kids' Collection with filters
- `/product/:id` — Product Detail with size selection, cart, reviews
- `/cart` — Cart + Checkout form
- `/login` — Login / Register
- `/account` — User Account (Profile, Orders, Contact)
- `/about` — Brand Story

## Features
- Fully responsive (mobile + desktop)
- Add to cart with size selection
- Quantity management
- Category & sort filters
- Smooth Framer Motion animations
- Gold & black luxury design system
- Playfair Display + Poppins typography
